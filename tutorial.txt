/ - корневая папка
cat - конкатинация (склеивание) читает файл
cat folder-name/file - читает file в папке folder-name
> - перезаписывает всё, что было в file
>> - дозаписать к file (дописать в конец)
cat folder-name > file - читает file в папке folder-name и записывает содержимое file
cp - копирование файлов
cp file1 copy-file1 - file1 (откуда), copy-file1 (куда)
cp -r folder-name copy-folder - скопировать из одной папки в другую (папка copy-folder создастся сама)
-s - мягкая ссылка (аналог ярлыка)
ln -s file1 soft-link-file1 - file1 (источник, на который ссылаются) soft-link-file1 (имя ссылки)
ln file2 hard-link-file2 - создание жесткой ссылки,которая ссылается непосредственно на сам файл
Если удалить файл, на который ссылаются мягкие ссылки - то эти ссылки станут битыми
Если удалить файл, на который ссылаются жесткие ссылки - то удалятется не сам файл, а обращение к нему
mv - перемещение файлов/папок
* - любое количество любых символов, включая ничего
[] - один любой символ из перечисленных в скобках
[0-9] - цифры
rm *[0-9] - удаляет все файлы, которые заканчиваются на цифры
rm file? - удаляет все файлы, которые начинаются на file
rm [fF]ile? - удаляет все файлы, которые начинаются на file/File
? - один любой символ
[!] - кроме перечисленного в скобках
[[:upper:]], или [A-Z] - любая большая буква
[[:lower:]], или [a-z] - любая маленькая буква
[[:digit:]], или [0-9] - любая цифра
[[:alnum:]], или [a-zA-Z0-9] - любая цифра, или буква
